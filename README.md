# House price Predictor
This project is a data-driven house price prediction tool built using R. It combines data visualisation and statistical analysis - Regression to analyze the housing data and predict prices based on property features.

FEATURES :

1. Data cleaning
2. Exploratory Data Analysis (EDA) on housing features.
3. Interactive visualizations with ggplot and plotly.
4. Multiple Linear Regression model for price prediction. 

DEPENDECIES :

1. ggplot2
2. plotly
3. dplyr
4. GGally
5. caret

DATASET :
https://www.kaggle.com/datasets/yasserh/housing-prices-dataset

The dataset is assumed to be a .csv file named "Housing.csv" containing the following columns
- price
- area
- bedrooms
- bathrooms
- stories
- mainroad (accessibility to mainroad)
- guestroom
- basement
- airconditioning
- parking
- prefarea
- furnishingStatus
